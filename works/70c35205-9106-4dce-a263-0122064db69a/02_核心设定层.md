# 核心设定层

## 稳定背景

- 官方 ChatGPT Plus 订阅价格较高，但二级市场中存在低价 Plus、Pro 月卡、Team 拼车、成品号、共享位、试用资格、企业池子和 API 中转等拆分售卖方式。
- 低价账号市场依赖上游渠道、账号池、异常订阅链路、试用资格与信息差，普通用户通常只关心能否登录和使用模型，很少追问来源。
- 市场话术长期围绕“低价稳定”“稳定质保”“官方同源”等承诺建立信任，但这些承诺在账号失效、订阅掉线、Team 炸号和售后不保时迅速坍塌。
- OpenAI 官方账号停用规则包括违反服务条款、绕过安全或访问限制、不当共享账号或 API key、破坏服务完整性等；文章将这些规则与灰色账号市场风险相连。

## 核心主题

- 灰色市场的利润来自信息不透明、风险后置和责任转移；赚钱时没人追问来源，亏损时才开始争夺“真相”。
- “风控”是文章中的关键遮蔽词：它既可能指平台规则执行，也可能被商家用来掩盖共享账号、异常订阅、试用资格滥用、上游跑路、同行举报等不同原因。
- Killeryou 在文章中不是稳定事实源，而是一个触发市场恐慌和叙事聚焦的名字；他的具体亏损、动机和举报行为都被标注为传言或未证实。
- 真正的戏剧性不在单个角色是否“吹哨”，而在整个低价 Plus 生意如何从“红利叙事”翻转成“风险暴露”。

## 使用边界

- 写作时可以保留悬疑、旁观者记录、行情表变化和社群语气变化等叙事手法。
- 涉及具体人物、平台漏洞和因果关系时必须使用“据说”“传言”“未证实”“市场反应显示”等限定语。
- 不应把“充值漏洞”“官方系统被攻破”写成确定事实；文章本身也强调这些说法被辟谣或无法确认。
